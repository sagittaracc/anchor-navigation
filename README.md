# Custom anchor navigation
### Include
```html
<script type="text-javascript" src="/path/to/anchor-navigation.js"></script>
```
### Usage
```js
navigation('/page.php')
    .anchor("#anchor-1", function(){
        // Do stuff on the anchor-1
    })
    .anchor("#anchor-2", function(){
        // Do stuff on the anchor-2
    })
```
